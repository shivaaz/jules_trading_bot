# Ultimate AI Trading Bot (Indian Markets, Legal Forex, Crypto)

This bot is designed to analyze local (Indian) and international news using Artificial Intelligence (Natural Language Processing via `vaderSentiment`) to generate `BUY`, `SELL`, or `HOLD` signals for specific legal assets in India.

## Features
- **News Scraping**: Aggregates top headlines from global finance, Indian markets, forex, and cryptocurrency sources via RSS feeds.
- **Sentiment Analysis**: Uses the VADER Sentiment Intensity Analyzer to score news sentiment from -1 (Negative) to +1 (Positive).
- **Signal Generation**: Maps sentiment scores to predefined thresholds to output actionable trading signals.
- **Legal Scope**: Specifically configured for:
  - Major Indian Equities/Indices (NIFTY 50, SENSEX, RELIANCE, etc.)
  - Permitted Forex Pairs traded on Indian Exchanges (USDINR, EURINR, GBPINR, JPYINR)
  - Major Cryptocurrencies (BTC, ETH, etc. - Note: Taxable under Indian VDA rules)

## Installation

1. Clone this repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the main application:
```bash
python main.py
```

The bot will perform an initial scan, analyze the latest news, generate signals, and then enter a scheduled loop (running every hour by default).

## Important Legal Disclaimers for Indian Traders

1. **Not Financial Advice**: The signals generated by this bot are entirely automated based on news sentiment and do not constitute professional financial advice. Always do your own research (DYOR).
2. **Forex Trading in India**: Retail forex trading via offshore brokers/platforms offering CFDs or margin trading on non-permitted pairs is illegal under the Foreign Exchange Management Act (FEMA), 1999. You must only trade permitted currency derivatives (like USDINR) through recognized Indian stock exchanges (NSE, BSE, MSEI) via SEBI-registered brokers.
3. **Cryptocurrency in India**: As of current regulations, cryptocurrencies are classified as Virtual Digital Assets (VDAs). Trading them is not illegal, but they are subject to strict taxation (e.g., 30% flat tax on profits + 1% TDS on transactions). They are not legal tender.
4. **Algorithmic Trading**: SEBI strictly regulates algorithmic trading in India. If you intend to execute trades automatically on Indian exchanges using this bot's signals, your trading algorithm must be approved by the exchange (NSE/BSE). Using unapproved APIs for live automated trading is a violation of SEBI rules.
